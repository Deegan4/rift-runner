# Rift Runner

A survivors-like / bullet-heaven game. Vanilla JS + HTML5 Canvas, no build step.

## Run it

```bash
npm start
```

Then play at `http://localhost:8080`. Or just open `index.html` in a browser (some browsers block ES modules from `file://` — use the server if so).

## Controls

- **WASD** or **Arrow keys** — move
- **`** (backtick / tilde) — toggle debug overlay
